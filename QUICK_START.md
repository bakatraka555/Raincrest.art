# 🚀 Quick Start - Raincrest.art

## ⚡ Brzi Start za Testiranje

### **Lokalno Testiranje (5 minuta)**

**1. Instaliraj jednostavni HTTP server:**
```powershell
npm install -g http-server
```

**2. Pokreni server:**
```powershell
cd C:\Users\bakat\Desktop\tapthemap\raincrest.art
http-server -p 8888 -a 0.0.0.0
```

**3. Pronađi IP adresu:**
```powershell
ipconfig
```
Traži **IPv4 Address** (npr. `192.168.1.100`)

**4. Otvori na mobitelu:**
```
http://192.168.1.100:8888
```

---

### **Deploy na Netlify (10 minuta)**

**1. Login:**
```powershell
netlify login
```

**2. Deploy:**
```powershell
netlify deploy --prod
```

**3. Otvori URL koji dobiješ na mobitelu!**
